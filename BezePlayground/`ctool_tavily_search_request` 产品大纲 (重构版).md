### `ctool_tavily_search_request` 产品大纲 (重构版)

#### 核心简要概括 (Executive Summary)

`ctool_tavily_search_request` 是 CoolCode 体系下受 SafeMode 严格管制的 Tavily 专属联网搜索工具。其核心设计理念为“本地优先、全程审计、结果落盘、隔离上下文”。

- **非默认能力**：仅当本地 CTool 无法满足需求时申请调用。
    
- **防上下文污染**：搜索结果强制以 Markdown 文件形式落盘到本地缓存区，工具仅向大模型返回“摘要”和“文件路径”，大模型需使用读取工具查看详情。
    
- **多层级配置与 Token 轮换**：系统级配置保管并轮换 API Key，角色级配置仅能收紧权限，杜绝密钥泄露风险。
    
- **四级风险门禁**：通过 Green（免审）、Yellow（单次确认/Confirm）、Red（双重确认/Double Confirm）、Blocked（硬拒绝/Reject）对请求进行拦截与审查。
    

#### 具体执行规范 (Execution Details)

##### 1. 配置与 Token 策略

- **系统级设定 (`LauncherDir\.cool-system\tavily.toml`)**：
    
    - 掌握最高权限，存储多组 `tokens`。
        
    - **轮换机制**：自动从第一个启用的 token 尝试，遇 429/401/403 自动切换下一个，全程对外（含日志和 AI 上下文）仅展示 Token 的 `name`（别名），绝不暴露 `api_key`。
        
- **角色级设定 (`CharacterRoot\.cool\config.toml`)**：
    
    - 仅能收紧权限（如降低最大搜索数、关闭特定功能），无法覆盖系统限制。如果此配置中出现 Token 字段，直接触发 Blocked 拦截。
        

##### 2. 四级风险门禁与确认交互

- **🟢 Green (普通文本搜索)**：如纯技术文档查询。免确认自动执行，但必须强制展示请求并记录日志。
    
- **🟡 Yellow (边界搜索)**：如宽泛查询、非官方网页提取。需要用户输入 `Y` 进行一次**确认 (Confirm)**，输入 `N` 或其他内容则**拒绝 (Reject)**。
    
- **🔴 Red (高危操作)**：如图片搜索、含敏感词（token/key/源码）的查询。强制要求两次**确认 (Confirm)**。
    
- **⚫ Blocked (硬拒绝)**：触发红线（下述）。直接拦截并记录，不进入确认流程。
    
- _交互增强_：用户在输入 `N` **拒绝 (Reject)** 时，可直接附带自然语言反馈（如“N 先查本地配置”），该反馈将无缝回传给大模型。
    

##### 3. 输出规范与日志落盘 (核心防线)

- **数据隔离**：绝对禁止将大段网页正文直接塞入大模型上下文。工具的返回值仅限：状态、简短摘要、下一步建议、以及生成的本地 MD 文件路径。
    
- **文件生成**：
    
    - 所有请求保存在只读保护目录：`CoolDir\cache\web_search\YYYY-MM-DD\`。
        
    - 统一命名规范：`0000X_<action>_<slug>.md`。
        
- **全局审计日志 (`request_log.md`)**：每次请求均追加记录时间、动作、风险等级、执行状态、Token别名及最终输出的文件路径。
    

##### 4. 支持的 Action 模式

1. **`search`**：标准网页搜索。
    
2. **`extract`**：提取指定 URL 的正文内容。
    
3. **`zoom`**：对已提取内容的特定部分进行深度聚焦。
    
4. **`research`**：综合多轮外部资料。
    
5. **`search_with_images`**：图片搜索（默认关闭，开启后强制视为 Red 风险，且仅允许 jpg/jpeg/png/webp 格式）。
    

##### 5. 绝对安全红线 (触发即 Blocked)

请求中出现以下任何意图，工具必须立即熔断：

- **密钥相关**：搜索、泄露、恢复或绕过 token、私钥、密码。
    
- **危险下载**：下载或保存可执行文件、脚本或压缩包（exe, dll, bat, ps1, sh, zip 等）。
    
- **动态执行**：试图打开系统浏览器、模拟浏览器行为、或执行网页 JavaScript。
    
- **代码越权**：要求上传本地私有文件全文，或执行 `git clone` 外部仓库。
